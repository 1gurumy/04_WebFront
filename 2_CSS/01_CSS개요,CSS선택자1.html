CSS(Cascading Style Sheets)란

      마크업언어(HTML)가 실제로 화면에 
      표시되는 방법(모양, 색상, 크기, 스타일..)을 기술하는 언어
      == HTML코드가 화면에 어떤 스타일로 보여질지 지정하는 것이 CSS

CSS Selector(선택자)

      선택자란?
      현재 HTML 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열.
      

      특정한 스타일을 적용하고자 하는 요소를 선택하는 방법

태그 선택자

      현재 HTML 문서 내에 같은 태그를 선택하는 선택자

      [작성법]

      태그명 {  (블록 내부가 태그 선택자)
        CSS코드;
      }
strong 태그에 작성된 내용
span태그에 작성된 내용
  • strong 태그에 작성된 내용
  • strong 태그에 작성된 내용

id 선택자

      HTML 문서 내에서 id 속성값이 일치하는 요소를 선택하는 선택자.

      * id 속성 : 요소를 식별할 수 있는 고유 식별자 역할
      한 HTML 문서 내에서 같은 id를 중복으로 작성 금지

      [작성법]

      #id속성값 {
        css 코드;
      }

      # == id

Class 선택자

      HTML 문서 내에서 같은 class 속성값을 가지는
      태그를 모두 선택하는 선택자

      [작성법]

      .class속성값 { css 코드; }

      .(마침표) == class

모든 요소 선택자(*)

      html의 모든 요소를 선택하는 선택자

      [작성법]

      * { css 코드; }
